# Cosmos networks

## Run install script

For mainnet
```bash
bash <(curl -s https://raw.githubusercontent.com/staketown/cosmos/master/composable/main_install.sh)
```

For testnet
```bash
bash <(curl -s https://raw.githubusercontent.com/staketown/cosmos/master/composable/test_install.sh)
```
